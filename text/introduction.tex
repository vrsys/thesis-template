Hypotheses and sub-hypotheses can be formulated like this:
\begin{hypothesis}
\label{hyp:test}
This is a hypothesis.
\end{hypothesis}
\begin{shypothesis}
\label{shyp:test}
This is a sub-hypothesis.
\end{shypothesis}
Hypothesis \cref{hyp:test} is very importance and must be referenced in the text.
